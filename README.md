# 🛍 AppleStore - MERN E-Commerce Website

A full-stack MERN E-Commerce web application developed as part of the CodeAlpha Full Stack Development Internship.

---

# 📌 Features

## User Features

- User Registration
- Email OTP Verification
- Login & Logout
- Forgot Password (OTP Based)
- Reset Password
- User Profile
- Edit Profile
- Product Listing
- Product Details
- Product Search
- Product Category Filter
- Product Price Sorting
- Shopping Cart
- Checkout
- Place Order
- Order History
- Product Review & Rating
- Responsive Design

---

## Admin Features

- Admin Login
- Dashboard
- Manage Products
- Add Product
- Edit Product
- Delete Product
- Manage Orders
- Update Order Status
- View Customer Feedback
- Responsive Admin Panel

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/dakshbhuva-78/CodeAlpha_Simple_Ecommerce_Store.git
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

EMAIL_USER=your_email@gmail.com

EMAIL_PASS=your_google_app_password
```
